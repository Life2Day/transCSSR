Python implementation of the Causal State Splitting Reconstruction (CSSR) algorithm for inferring epsilon-transducers from data generated by discrete-valued, discrete-time input/output systems.
https://github.com/ddarmon/transCSSR

Install Notes:

Python – Version 2.7

Igraph – use igraph(0.7.1). The igraph(0.1.11) is not the right version.

    python-igraph 0.7.1.post6
    https://pypi.python.org/pypi/python-igraph

    Available pre-complied libraries. Unofficial Windows Binaries for Python Extension Packages
    https://www.lfd.uci.edu/%7Egohlke/pythonlibs/

    Example: 64bit-python_igraph
    python_igraph-0.7.1.post6-cp27-cp27m-win_amd64.whl
    Download the file and manually install with pip.
