o All print functions need to be convert from print ‘text’ to print(‘text”) using brackets.
o All dictionary references using keys in loops, for example, 'deepcopy'.keys() will need to be converted being
you will get 'can't pickle dict_keys objects' because Python 3 returns iterable view.
